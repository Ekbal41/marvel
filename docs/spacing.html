#Paddings

'p-1' to 'p-10'. classes are available of padding.

For responsiveness you can use breakpoints like 'p-md-1'.

For directional paddings use 'p[x]-1'.

Here 'x' can be 'l' 'r' 't' 'b' as for left-rignt use 'px-1' and for top-bottom use 'py-1'

#Margins

'm-1' to 'm-10'. classes are available of margin.
For responsiveness you can use breakpoints like 'm-md-1'.

For directional margins use 'm[x]-1'.

Here 'x' can be 'l' 'r' 't' 'b' as for left-rignt use 'mx-1' and for top-bottom use 'my-1'
